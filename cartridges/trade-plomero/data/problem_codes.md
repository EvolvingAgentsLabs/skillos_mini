# Problem codes — trade-plomero

| Code                                | Severidad | Cuándo se aplica                                                  |
|-------------------------------------|-----------|-------------------------------------------------------------------|
| `perdida_presion_activa`            | 5         | Goteo o chorro continuo desde cañería de presión                 |
| `perdida_presion_intermitente`      | 4         | Mancha que crece, no goteo continuo (presión con pequeña fisura)  |
| `filtracion_descarga_piso_superior` | 3         | Mancha en techo correlacionada con uso de artefacto arriba         |
| `obstruccion_desague`               | 3         | Desagüe lento, retorno de agua, posible olor                       |
| `obstruccion_total`                 | 4         | Desagüe completamente bloqueado                                    |
| `inodoro_perdida_base`              | 3         | Junta de cera/wc fallando — agua filtra alrededor de la base       |
| `flexible_oxidado`                  | 3         | Flexible de conexión con oxidación visible o filtración            |
| `valvula_paso_dura_o_pierde`        | 2         | Llave de paso atascada o gotea por el vástago                      |
| `tanque_filtracion_visible`         | 4         | Tanque con grieta o pérdida visible                                |
| `bomba_no_arranca`                  | 4         | Presurizadora con falla eléctrica o mecánica                       |
| `presion_baja`                      | 2         | Presión insuficiente en piso superior — diagnóstico de causa abierto |
| `humedad_estructural_pendiente`     | 3         | Humedad recurrente compromete viga/entrepiso (escalar a estructura) |
| `cañeria_galvanizada_obsoleta`      | 2         | Cañería de hierro galvanizado vieja, todavía no falla pero al límite |
| `sello_artefacto_dañado`            | 2         | Junta de ducha / lavabo / bidet en falla                           |
| `desnivel_desague`                  | 3         | Desagüe sin pendiente suficiente (<1%)                              |
