# Problem codes — trade-pintor

| Code                          | Severidad | Cuándo se aplica                                                |
|-------------------------------|-----------|-----------------------------------------------------------------|
| `pintura_descascarada`        | 3         | Pintura existente con escamas, descascarado parcial o total     |
| `humedad_localizada`          | 3         | Manchas de humedad puntuales sin signo activo                  |
| `humedad_estructural`         | 4         | Humedad activa o creciente — escalar antes de pintar           |
| `moho_visible`                | 3         | Hongo en superficie — requiere fungicida antes de tratamiento   |
| `oxido_metal`                 | 3         | Carpintería metálica con óxido — lijado + convertidor            |
| `yeso_dañado`                 | 2         | Yeso con grietas / agujeros menores — empaste                   |
| `pintura_vieja_oleosa`        | 2         | Pintura existente al aceite — requiere lijado fuerte + fijador  |
| `superficie_lisa_lista`       | 1         | Lista para pintar con preparación mínima                         |
| `cielorraso_manchas_humedad`  | 3         | Manchas en cielorraso — investigar origen antes de pintar       |
| `madera_carcomida`            | 4         | Carpintería de madera dañada — requiere reparación previa       |
| `panel_yeso_quebrado`         | 3         | Drywall con perforaciones / golpes                               |
