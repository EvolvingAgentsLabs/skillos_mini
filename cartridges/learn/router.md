# Learn Cartridge Router

Classify the user's goal into one of these flows:

- **learn**: User wants to learn about a topic (research + store)
- **research**: User wants quick information without storing
- **quiz**: User wants to test their knowledge ("quiz me", "test me")
- **review**: User wants to recall stored knowledge ("what do I know about")
- **stats**: User wants learning statistics ("stats", "progress", "what have I learned")

Reply with ONE WORD: the flow name.
